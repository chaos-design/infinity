# 行为准则

## 我们的承诺

为了营造开放、友好、专业的协作环境，Infinity 项目欢迎不同背景、经验水平和观点的参与者。我们承诺为所有贡献者和使用者提供免受骚扰的社区体验。

## 鼓励的行为

- 使用友善、包容和尊重的语言。
- 接受建设性反馈，并关注问题本身。
- 在讨论中给出清晰上下文、复现信息和可执行建议。
- 尊重维护者和贡献者的时间，保持 Issue 与 Pull Request 聚焦。
- 对社区其他成员表现出同理心。

## 不被接受的行为

- 人身攻击、侮辱、贬低或挑衅。
- 基于身份、背景、技术水平或观点的歧视性表达。
- 发布他人隐私信息或敏感信息。
- 持续打断讨论、重复刷屏或刻意偏离主题。
- 任何形式的骚扰、威胁或不专业行为。

## 维护者职责

维护者有责任解释并执行本行为准则。对于不符合准则的内容，维护者可以采取编辑、隐藏、关闭、锁定讨论、限制参与或其他必要措施。

## 适用范围

本行为准则适用于项目仓库、Issue、Pull Request、Discussions、代码评审以及与项目相关的公开或私下交流。

## 反馈与处理

如果你看到或经历不当行为，请通过项目维护者可用的私密渠道反馈。维护者会尽力以尊重隐私和事实核查的方式处理。
